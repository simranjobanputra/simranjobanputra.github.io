Project Overviews

As a summmer UX designer at NASA AMES, I was assigned to two main projects. One of my projects was creating hotkeys and shortucts for Playbook, an iPad application that astronauts can use to edit a mission while on the mission. This project involved understanding how astronauts would use this application and how we could reorganize the interface to make the most sense. The second project I worked on was redesigning social page for NASA HUNCH. NASA HUNCH is the website used to recruit high school students for projects on the International Space Station.
